Các lệnh cơ bản:

- <h1>…</h1>, <h2>…</h1>,…, <h6>…</h6>: tiêu đề theo kích cỡ giảm dần
- <p>…</p>: viết nội dung của đoạn văn bản (có sẵn space ở trước và sau đoạn
- <br>: ngăn cách dòng
- <hr>: tạo đường kẻ ngang
- <a>…</a>: tạo liên kết
    - Cấu trúc:
        <a href=”link” target=_blank/_self title=”Nội_dung_khi_trỏ_vào_link”> 
          Tên_liên_kết
        </a> 
- <ul>
    <li>…</li>
  </ul>
    → Danh sách ko sắp xếp
- <header>…</header>: block để chứa ndung giới thiệu
- <aside>…</aside>: block để chứa ndung phụ
- <blockquote>…</blockquote>: block lời trích dẫn
- <q>…</q>: 1 dòng trích dẫn
- <code>…</code>: đánh dấu phần nội dung bên trong là code với font chữ khác

Format:

- <b>…</b>: in đậm
- <strong>…</strong>: in đậm nhưng nhấn mạnh ndung
- <i>…</i>: in nghiêng
- <em>…</em>: in nghiêng nhưng nhấn mạnh ndung
